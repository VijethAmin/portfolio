Bank Loan Analysis Dashboard

I built a Bank Loan Analysis Dashboard in Power BI to analyze loan performance and risk.

I used Power Query for data cleaning and DAX for calculations.

The dashboard includes KPIs like total applications, funded amount, and received amount, along with loan classification into good and bad loans.

I also created monthly trend analysis and interactive filters based on loan grade, purpose, and region.

This project helps in understanding profitability, risk exposure, and customer behavior, enabling better decision-making for financial institutions.


Indian Stock Market Dashboard

I developed an Indian Stock Market Dashboard using Streamlit that provides real-time stock analysis, forecasting insights.

The application fetches stock data using yfinance, visualizes it with interactive charts, and applies technical indicators like RSI, MACD, SMA, and EMA for analysis.

For forecasting, I implemented multiple time-series models such as ARIMA, SARIMA, Prophet, and LSTM, and evaluated their performance using metrics like MAE and RMSE.

Additionally, I designed a scalable data pipeline deployed using AWS EC2 and S3 for storage and Athena for query with IAM roles for secure querying and data export.


House Price Prediction - ML Web App


“I built a machine learning web app to predict house prices in Bengaluru using a Kaggle dataset. It includes features like location, size, bedrooms, and bathrooms.

I cleaned the data by handling missing values, converting ranges into numbers, and removing outliers. I also created useful features like BHK and encoded location data.

I tried different models like Linear Regression, Random Forest, and XGBoost, and chose Linear Regression because it gave the best accuracy.

Finally, I deployed the model using Flask, where users can enter house details and get predicted prices instantly.”
